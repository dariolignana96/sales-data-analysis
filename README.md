# Sales Data Analysis

Analisi esplorativa di dati di vendita con Python. Progetto dimostrativo di competenze in **data manipulation**, **visualization** e **analytical thinking**.

## Obiettivi

- Caricamento e analisi dati con pandas
- Visualizzazione dati con matplotlib e seaborn
- Creazione di un report visuale semplice

## Struttura del progetto

├── data/ # Dataset usati nel progetto
├── scripts/ # Script Python per analisi dati
├── notebooks/ # Jupyter notebook interattivi
├── visualizations/ # Grafici e output visivi
└── README.md # Documentazione progetto


## Come eseguire

1. Clona la repository
2. Installa dipendenze con:

pip install pandas numpy matplotlib seaborn



3. Esegui lo script di analisi:

python scripts/analysis.py



## Risultati

- Generazione di grafici di vendita per prodotto
- Salvataggio report visuale in visualizations/
